# **App Name**: Eider Insights

## Core Features:

- Data Fetch: Fetch campaign analytics data from the provided webhook URL, filtering by start and end dates.
- Campaign Overview: Display a high-level overview of all campaigns, including key metrics like leads count, contacted count, and emails sent.
- Campaign Details: Provide a detailed view for each campaign, showing all available metrics (campaign ID, status, evergreen status, counts for leads, contacted, open, reply, link click, bounced, unsubscribed, completed, emails sent, new leads contacted, total opportunities, total opportunity value).
- Date Filtering: Implement a UI component to filter campaign data by start and end dates, updating the dashboard in real-time.

## Style Guidelines:

- Primary color: Deep teal (#008080), for a professional and trustworthy feel, reflecting insight.
- Background color: Light cyan (#E0FFFF), providing a clean, unobtrusive backdrop to focus attention on the data visualizations.
- Accent color: Soft turquoise (#7DF9FF), used sparingly for highlights and interactive elements to draw the eye without overwhelming the design.
- Body and headline font: 'Inter', a grotesque sans-serif for a clean, modern, readable style appropriate to a dashboard.
- Modern dashboard layout with clear sections for overall statistics and detailed campaign breakdowns.
- Use simple, line-based icons to represent different metrics, providing quick visual cues.
- Subtle transitions and animations when data is filtered or updated to enhance user experience.
